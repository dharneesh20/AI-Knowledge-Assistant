# AI Knowledge Assistant

## Description
Developed an AI Knowledge Assistant using Python and Django. The system allows users to upload documents (PDF, DOCX, TXT) and ask questions based on uploaded content. It uses embeddings, FAISS vector database, and Groq LLM for semantic retrieval and intelligent responses.

## Features
- User Authentication
- PDF/DOCX/TXT Upload
- Text Extraction
- Chunking
- Embeddings Generation
- Vector Search using FAISS
- AI Answer Generation using Groq API

## Technologies Used
- Python
- Django
- SQLite
- FAISS
- Sentence Transformers
- Groq API
- HTML/CSS

## Project Workflow

User Uploads Document
↓
Text Extraction
↓
Chunking
↓
Embeddings Creation
↓
Store in FAISS
↓
User Query
↓
Similarity Search
↓
Groq LLM Response

## Installation

Clone repository:

git clone REPOSITORY_URL

Install dependencies:

pip install -r requirements.txt

Run server:

python manage.py runserver
